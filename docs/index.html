Awesome Seedance 2 Prompts

A curated, open-source library of prompts for ByteDance's Seedance 2.0 video model — built and maintained by Martini Art. For the full editorial walkthrough with embedded videos, see the original article on Martini Art.

Read the companion guide: Seedance 2 Handbook — variants, strengths, and the production workflows that fit on Martini's canvas. Or skim the handbook landing page.

What's inside

Seedance 2.0 is one of the strongest text-to-video and image-to-video models released in 2026. This repository collects production-tested prompt recipes that ship clean motion, stable physics, and consistent characters across shots. Each prompt entry contains the exact text, recommended parameters, the source image (when image-to-video), and a generated example clip.

  • Product ad prompts — 360° spins, surface materials, packshots, lifestyle motion
  • Image-to-video prompts — bring stills into motion with stable subject identity
  • Cinematic camera moves — dolly, push-in, parallax, orbit, crane
  • Character continuity — multi-shot scenes with the same face and wardrobe
  • Physics-rich motion — fluids, fabric, hair, light interaction

Run the prompts

Every prompt in this collection runs unchanged on the Martini Art canvas, which orchestrates Seedance 2.0 alongside Veo 3.1, Kling 3.0, Soul V2, and Nano Banana 2 in a single timeline. The canvas handles seed continuity, batch variations, and stitching multi-shot scenes. You can also paste the prompts directly into ByteDance's official Seedance API.

FAQ

What is Seedance 2?

Seedance 2.0 is ByteDance's flagship video generation model. It produces text-to-video and image-to-video clips with strong physics, controllable camera moves, and stable subject identity across shots.

How do I run these prompts?

The fastest path is the Martini Art canvas — paste a prompt and it runs on Seedance 2.0 plus a side-by-side comparison against Veo 3.1 or Kling 3.0 if you want. Power users can call the official Seedance API directly.

Are the prompts free to use?

Yes. The repository is open-source. Prompts are licensed for reuse with attribution to Martini Art.
