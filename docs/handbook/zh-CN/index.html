版本一览

版本最适合取舍
Seedance 2 Pro英雄镜头、终稿、客户交付最慢、每秒成本最高
Seedance 2 Lite迭代、A/B 提示词测试、批量变体边缘运动略软
Seedance 2 i2v图生视频 — 让静态图动起来质量受源图限制

Seedance 2 擅长什么

  • 物理 — 布料、液体、头发、接触行为合理
  • 运镜 — 推、拉、环绕、视差、升降都能稳定跟随提示词
  • 主体一致性 — 人脸、产品、角色在整段片中保持稳定
  • 光影一致 — 曝光与色温整段稳定

生产工作流

在 Martini Art 画布 上推荐的流程:

  1. 从 Awesome Seedance 2 提示词合集 选一个配方起草
  2. 用 Seedance 2 Lite 跑一轮快速批量(4–8 个变体)锁定运动方向
  3. 可选:在画布上把同一条提示词并行跑 Veo 3.1 或 Kling 3.0 做横向对比
  4. 把赢的种子升级到 Seedance 2 Pro 出终稿
  5. 用 Soul V2 处理人物一致性,在画布时间线上拼接多镜头

常见问题

应该用哪个 Seedance 2 版本?

终稿用 Pro,迭代用 Lite,有源图做动态用 i2v。详情见上方对比表。

Seedance 2 支持图生视频吗?

支持。i2v 能保持主体身份稳定,同时按提示词执行运镜与运动。配方在 提示词合集 里。

API 在哪里获取?

字节跳动通过官方 API 提供 Seedance。Martini Art 把它和 Veo 3.1、Kling 3.0、Soul V2、Nano Banana 2 打包在画布上,免写 API 代码即可对比与拼接。
